# Todo API

Simple REST API for task management built with Flask and SQLite.

## Features

- Create tasks
- Get all tasks
- Get task by id
- Update tasks
- Delete tasks
- Search tasks
- Filter by completed
- Sorting
- Pagination

## Technologies

- Python
- Flask
- SQLite
- Git
- GitHub
